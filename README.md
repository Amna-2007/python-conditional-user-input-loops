# Python Basics Project

## Description
Practice repository for Python basics: Conditional statements, User input, and Loops. This project shows my learning journey in Python programming.

- User Input
- Loops (for/while)

## Example
```python
# Conditional example
x = 10
if x > 5:
    print("x is greater than 5")

# User Input example
name = input("Enter your name: ")
print("Hello", name)

# Loop example
for i in range(1, 6):
    print("Iteration:", i)
